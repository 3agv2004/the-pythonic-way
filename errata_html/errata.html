Pagina 148¶

Segnalato da Gollum1

Originale¶

>>> any([]) # Restituisce True se l'oggetto iterabile e' vuoto

La funzione a cui si fa riferimento e’ all() non any(). Inoltre manca il risultato.

>>> all([]) # Restituisce True se l'oggetto iterabile e' vuoto
True

Pagina 151¶

Segnalato da Gollum1

Originale¶

>>> sorted([(1, 2, 3), 'python', {'quattro': 4, 'cinque': 5, 'sei': 6, 'otto': 8}],
... key=len, reverse=True)
['python', {'cinque': 5, 'quattro': 4, 'otto': 8, 'sei': 6}, (1, 2, 3)]
['python', {'sei': 6, 'cinque': 5, 'otto': 8, 'quattro': 4}, (1, 2, 3)]}}}

Sono stati riportati due output piuttosto che uno.

Corretto¶

>>> sorted([(1, 2, 3), 'python', {'quattro': 4, 'cinque': 5, 'sei': 6, 'otto': 8}],
... key=len, reverse=True)
['python', {'sei': 6, 'cinque': 5, 'otto': 8, 'quattro': 4}, (1, 2, 3)]}}}
